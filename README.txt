[各ディレクトリの役割]
- TSGenerator: 時系列データの生成。
- CPSPEC     : パワースペクトル・クロススペクトルの計算。
- PHAGEN     : XSPEC読み込み用ファイル作成。

[即時的な動作方法]
>>> cd /path/to/TSGenerator
>>> python commands.py
>>> cd /path/to/CPSPEC
>>> python commands.py
>>> cd /path/to/PHAGEN
>>> python commands.py

[簡単な説明]
parameters.pyを編集することで、簡単なパラメータの変更が可能。
